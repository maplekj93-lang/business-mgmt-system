# L&D ERP 구현 현황 분석 리포트 (Gap Analysis)

V1.1부터 V2.7까지의 명세서(`L_and_D_ERP_Specification_V2.7.md`)와 현재 코드베이스(`src/`)를 대조하여, **구현된 부분과 방향이 다르거나 누락된 디테일**을 분석한 결과입니다.

---

## 🟩 1. 대체로 잘 구현된 모듈 (Work Mode 핵심 기능)

### A. 프로젝트 및 파이프라인 관리 (Project View)
- **명세 사양:** 단계별 칸반 보드 관리(대기/실행/정산/종료), 매출 확정(match-income 로직 연동).
- **구현 현황:** `KanbanBoard.tsx`를 통해 Drag & Drop 방식의 칸반 뷰가 잘 구현되어 있습니다. 실시간 파이프라인 가치를 보여주는 통계 카드도 적용됨.
- **💡 보완점 (Gap):** 
  - `Pre - Pro - Post` 단계에 특화된 아내분의 '디자인 기여도 투적(Layer 분리)' 로직이 파이프라인 자체에는 아직 명확히 녹아있지 않습니다.
  - 앱-퍼스트 동기화(구글 캘린더 자동 생성/동기화) 기능이 `KanbanBoard`나 `Project` 생성 스텝에 연결되어 있지 않습니다.

### B. 크루 일급 및 대장 관리 (Daily Rate)
- **명세 사양:** 크루 지급 내역, 원천징수, 통장 대조 검증, 순수익 계산.
- **구현 현황:** `LogDailyRateModal`, `DailyRateTable` 등을 통해 크루 인건비 지급 기록 및 상태 값(`paid`, `pending`) 수정이 완벽히 지원됨.
- **💡 보완점 (Gap):**
  - **은행 API 대조(Bank-Statement 대조):** 앱 내 '지급 완료' 처리와 실제 통장 출금을 대조하는 `Bank Sync Verifier` 로직은 미구현 상태입니다.

## 🟨 2. 구현은 되었으나 방향/디테일 보완이 필요한 모듈

### A. 가계부 및 금액 대시보드 (Dual View & Dashboard)
- **명세 사양:** Project View와 Cash Flow View의 완전한 듀얼 뷰, 3개월 버퍼(Safety Net) 시각화, 지출 레이어 세분화.
- **구현 현황:** `Home` 대시보드에서 좌측(가계/Family)과 우측(사업/Business)으로 50:50 분리된 뷰를 제공 중.
- **💡 보완점 (Gap):** 
  - **버퍼 가동률 (3개월치 현금 버퍼):** 안전망(Safety Net) 잔고를 %로 보여주는 시각화 로직이 대시보드에 누락되어 있습니다.
  - **가상 월급제 로직:** 사업 통장에서 가계 통장으로 이체되는 '고정액(가상 월급)'을 어떻게 처리하고 대시보드에서 차감/합산할지에 대한 로직이 없습니다.

### B. 세무(VAT) 격리 및 고정비 (Tax & Recurring)
- **명세 사양:** 부가세 10% 외 종소세(15~20%) 2중 격리. 구독 서비스의 사업비 매입세액 공제 시각화.
- **구현 현황:** `VatReserveCard`를 통해 부가세 분리 모니터링 적용 완료. `RecurringExpense`로 고정비 캘린더 적용 완료.
- **💡 보완점 (Gap):**
  - **종소세 격리 박스:** 부가세 외에 15~20%의 종합소득세를 두 번째 세금 박스로 떼어놓는 로직이 아직 UI와 DB에 없습니다.
  - 고정 지출 중 '사업용(Business)'과 '개인용(Personal)'을 캘린더나 내역에서 색상 등으로 엄격하게 레이어 분리해 주는 UI가 아직 부족합니다.

## 🟥 3. 미구현 / 신규 개발이 필요한 핵심 모듈

### A. 바터(Barter) & 크레딧 처리 로직
- **디테일:** 장비 렌탈비와 창고/사무실 월세를 상계(퉁치는) 거래.
- **상태:** 완전 미구현. 단순 지출/수입이 아니라 **`[운영비 절감액]`**으로 별도 표기하여 실제 통장 잔고 통계가 망가지지 않게 하는 특수 회계 처리 기능 모듈이 필요합니다.

### B. 스마트/하이브리드 자동 태깅 (Smart Tagging)
- **디테일:** 캘린더 장소/시간과 엑셀 카드 결제 내역을 교차 검증하여 식대, 주차비 등을 자동 분류 (현장, 네트워킹, 고정 운영비 등 추천). 영수증 OCR 기능 결합 포함.
- **상태:** `ledger-import`를 통해 엑셀을 업로드하는 기초 기능은 있으나, 알고리즘을 통한 '지능형 자동 태깅' 시스템 및 이중 지출 차단(사업비 전환 시 개인 가계부 자동차감) 로직이 없습니다.

### C. 장비 수익성(ROI) 및 모바일 체크리스트 
- **디테일:** 타사 장비 지분 기반 수익 창출, 자사 장비 투자 대비 감가상각 가시화, 현장 QR 분실 방지 체크 기능.
- **상태:** 장비(자산) 관리 `entities/asset` 껍데기만 존재할 뿐 UI나 API 로직 구현 안 됨.

### D. 상황 인지형 UI 및 워크라이프 밸런스 로직 (Life/Growth Mode)
- **디테일:** 현장이 있는 날(Field Mode)과 없는 날(Growth Mode)의 UI 동적 변화. '미래 투자 가치' 시각화, 아내와의 홈 앱스(루틴 확인, 소소한 리워드) 보상 체계, 비전 보드 화면.
- **상태:** 대시보드는 현재 정적인 재무 뷰만 있음. 자기계발 진도율이나 워크-라이프 컨텍스트와 연동된 모드 스위칭 불가능.

---

## 🚀 제안하는 다음 마일스톤 (택 1)

1. **[재무 디테일 완벽화]**: 종소세(15~20%) 예비비 박스 추가, 3개월 Safety Net 버퍼 UI 대시보드 연동, 가상 월급제 연동 설계하기.
2. **[특수 회계(바터) 도입]**: 사무실/장비 상계 거래를 실 잔고 왜곡 없이 '운영비 절감 레이어'로 처리하는 로직 추가.
3. **[스마트 태깅 & 영수증]**: 엑셀 장부와 구글 캘린더(일정/위치)를 교차 매핑하여 지출 내역을 자동 분류/추천해 주는 AI 추천 매핑 화면 설계.
4. **[성장 및 라이프 모드]**: 현장이 없는 날(Growth Mode) 대시보드에 비전보드, 자기계발 시급(가치 환산), 집안일 퀘스트 리워드 기능을 띄워주는 UI 제작.
