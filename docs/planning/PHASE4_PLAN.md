# 🚀 Phase 4 ERP 완성 및 기술 부채 청산 계획

## 🎯 개요
비즈니스 프로젝트 관리 및 정산 시스템을 완성하고, 기반 시스템을 고도화하여 데이터 무결성을 보장합니다.

---

## 🏗️ 1. 기반 시스템 (완료)
- **배치 관리:** `import_batches` 테이블을 통한 임포트 이력 관리.
- **중복 방지:** 해시 기반 멱등성 보장 시스템 구축.
- **데이터 정리:** 기저 중복 데이터 정리 및 레거시 데이터 해싱 작업 완료.

## 💼 2. 비즈니스 ERP 기능
- **견적서 시스템:** `CreateQuoteDialog` 및 `QuoteTemplate` (완료).
- **사업 지출 태깅:** 가계부 내역과 프로젝트 원가 연결 UI (진행 중).
- **수입 매칭:** 파이프라인 카드와 실제 입금 내역 연동 (진행 중).
- **캐시플로우 캘린더:** 프로젝트 수입/지불 일정 통합 위젯 (진행 중).

## 🛠️ 3. 기술 부채 및 최적화
- **실무 효율화:** 크루 원천징수 3.3% 자동 계산, 부가세 10% 자동 추적 (완료).
- **가계부 최적화:** 월정액 구독 관리 및 자동 기록 (완료).
