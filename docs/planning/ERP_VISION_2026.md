# 🎯 프로젝트 비전: 쾅영부부 통합 재무 OS (Phase 0)

이 문서는 프로젝트의 근본적인 목적과 핵심 가치를 정의하며, 모든 개발 의사결정의 기준이 됩니다.

## 1. 프로젝트의 사명 (The Mission)
"프리랜서 부부의 **일(Work)**과 **돈(Money)** 사이의 마찰력을 0으로 만든다."

단순히 기록하는 도구를 넘어, 현장에서 발생한 노동이 실제 통장의 숫자로 변환되고, 이것이 다시 부부의 자산과 세금 데이터로 자동 연결되는 **통합 재무 운영체제(Financial OS)**를 구축합니다.

## 2. 3대 핵심 기둥 (Three Pillars)

### 🟦 기둥 1: 초고속 인풋 (Capture)
- **현장 일당 기록기**: 현장에서 즉시 '광준 일당 + 크루비 + 경비'를 10초 만에 입력.
- **은행/카드 동기화**: CSV/엑셀 임포트를 통해 모든 지출 내역을 누락 없이 확보.
- **견적/청구**: 작업 완료 시 즉시 PDF 견적서 발행 및 파이프라인 등록.

### 🟩 기둥 2: 지능형 정산 (Reconcile)
- **매입/매출 매칭**: 통장 입금 내역과 프로젝트 잔금을 매칭하여 '미수금' 자동 해결.
- **사업비 태깅**: 가계부 지출 중 '이건 현장 경비야'라고 태깅하여 프로젝트 원가로 귀속.
- **더치페이**: 공동 비용을 명확히 정산하여 부채 관계 해소.

### 🟪 기둥 3: 전략적 리포트 (Insight)
- **실전 순이익**: 부가세(10%)와 원천세(3.3%)를 제외한 '진짜 내 돈'을 실시간 확인.
- **세무 준비**: 분기별 부가세 신고액 예측 및 소득세 대비 가이드 제공.
- **통합 자산**: 개인 예적금 + 사업 미수금 + 투자를 합산한 부부의 실질 순자산 추적.

## 3. 유저의 페인 포인트와 해결책 (Pain & Solution)

| 페인 포인트 (Pain) | 해결책 (Solution) |
|---|---|
| "내가 이번 달에 정확히 얼마 벌었는지 모르겠다." | 세전/세후/경비를 구분한 실시간 손익 대시보드 |
| "미수금이 어디 업체에서 얼마 남았는지 헷갈린다." | 상태별 칸반 보드와 입금 매칭 자동 알림 |
| "세금 신고 때마다 엑셀 정리하기 너무 힘들다." | 부가세/원천세 데이터가 자동으로 집계된 리포트 출력 |
| "가계부 쓰기가 귀찮고 자꾸 밀린다." | 은행 내역 임포트와 키워드 기반 자동 분류 엔진 |

## 4. 웻지(Wedge) 전략: 단계별 로드맵

- **Phase 1 (Logger)**: [현재 완료] 매일 발생하는 일당 데이터를 안전하게 보관.
- **Phase 2 (Pipeline)**: [다음 단계] 미수금을 관리하고 통장 내역과 연결.
- **Phase 3 (Tax & Insight)**: 축적된 데이터를 바탕으로 세금 예측 및 통합 자산 뷰 완성.
