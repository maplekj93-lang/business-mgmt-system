## 📋 **언제 어느 도구를 사용할까?**

### **상황 1: 카드사 내역에 "카카오페이"로만 표시됨**
```
카드사: "카카오페이" 10,000원
→ 사용할 도구: 카카오페이 매퍼 ✓

왜?
  - 가맹점 정보가 완전히 숨겨져 있음
  - 카카오페이 거래내역서로 보충 가능
  - 자동 매칭으로 간단하게 해결
```

### **상황 2: 카드사 내역에 "쿠팡"으로 표시됨, 하지만 여러 상품 구매**
```
카드사: "쿠팡" 50,000원 (노트북 + 우유 + 선물)
→ 사용할 도구: 쿠팡 분해기 ✓

왜?
  - 가맹점은 명확하지만(쿠팡) 상품이 섞임
  - 사업용(노트북)과 개인용(우유) 섞여있음
  - 쿠팡 주문서로 각각 분리 필요
```

### **상황 3: 카카오페이 + 쿠팡 동시 발생**
```
카드사: "카카오페이" 50,000원 (실제: 쿠팡에서 주문)
→ 사용할 도구: 카카오페이 매퍼 + 쿠팡 분해기 (둘 다!)

Step 1: 카카오페이 매퍼
  "카카오페이" 50,000원 → "쿠팡" 50,000원 (가맹점 명확화)

Step 2: 쿠팡 분해기 (선택, 필요시)
  쿠팡 주문서 있으면 상품별로 추가 분해
  없으면 쿠팡 전체를 1개 거래로 처리
```

### **상황 4: 카카오페이 + 배달 음식 여러 개**
```
카드사: "카카오페이" 15,000원
카카오페이 거래내역서:
  ├─ 배민 7,000원
  ├─ 우버이츠 5,000원
  └─ 쿠팡이츠 3,000원

→ 사용할 도구: 카카오페이 매퍼 ✓

결과: 3개 거래로 자동 분리
  ├─ [카카오페이] 배민 7,000원 → 식비 > 배달
  ├─ [카카오페이] 우버이츠 5,000원 → 식비 > 배달
  └─ [카카오페이] 쿠팡이츠 3,000원 → 식비 > 배달
```

## 🔄 **결정 트리 (Decision Tree)**

```
카드사 내역에서 시작
  │
  ├─ "카카오페이"로 표시?
  │  YES → 카카오페이 매퍼 사용
  │        (거래내역서로 가맹점 추출)
  │  NO  ↓
  │
  ├─ "쿠팡"으로 표시?
  │  YES → 쿠팡 분해기 필요?
  │        │
  │        ├─ 상품 여러 개 + Track A/B 섞임?
  │        │  YES → 쿠팡 분해기 사용
  │        │  NO  → 그냥 1개 거래로 처리
  │        │
  │        └─ 끝
  │  NO  ↓
  │
  └─ 기타 가맹점
     → 그냥 가계부에 직접 입력 (분류기 불필요)
```

## 💾 **데이터 흐름 (Data Flow)**

### 쿠팡 분해기
```
[카드사 내역] → [쿠팡 주문서] → [분해 도구]
  "쿠팡"        업로드          "쿠팡" 50,000원
  50,000원                      ├─ 노트북 30,000원 [비즈니스] (X)
                                ├─ 우유 5,000원 [생활]
                                └─ 선물 15,000원 [경조]
                                   ↓
                            [가계부에 3줄 기록]
                            (원본 1줄은 숨김)
```

### 카카오페이 매퍼
```
[카드사 내역] → [카카오페이 거래내역서] → [매핑 도구]
  "카카오페이"   CSV 다운로드              "카카오페이" 10,000원
  10,000원       (거래일, 금액, 가맹점)    ├─ 쿠팡 7,000원
                                        └─ 배민 3,000원
                                           ↓
                                    [가계부에 2줄 기록]
                                    (카카오페이는 숨김)
```

## ✅ **체크리스트: 올바르게 사용했나?**

### 쿠팡 분해기 사용 후
- [ ] 원본 "쿠팡" 거래는 숨겨졌는가?
- [ ] 분해된 N개 항목이 모두 보이는가?
- [ ] 각 항목이 다른 카테고리로 분류되었는가?
- [ ] 사업용 항목에 "비즈니스" 뱃지가 있는가?
- [ ] 합계가 원본과 일치하는가?

### 카카오페이 매퍼 사용 후
- [ ] 원본 "카카오페이" 거래는 숨겨졌는가?

- [ ] 각 항목에 실제 가맹점명이 표시되었는가? (예: "[카카오페이] 쿠팡")
- [ ] 각 항목이 적절한 카테고리로 분류되었는가?
- [ ] 합계가 원본 카드 결제액과 일치하는가?

## 🚨 **주의사항**

### 쿠팡 분해기
- ❌ 쿠팡 주문서 없이 분해하면 안 됨 (부정확함)
- ❌ 카카오페이로 결제했으면 먼저 카카오페이 매퍼 사용
- ✅ 분해 후 "비즈니스" 항목을 꼭 체크할 것

### 카카오페이 매퍼
- ❌ 거래내역서 없이 매칭하면 안 됨 (자동 기능 불가)
- ❌ 거래일과 금액이 다르면 매칭이 안 될 수 있음 (±1일 허용)
- ✅ 남은 머니가 있으면 경고 표시됨 (다음 거래와 합산 예상)

## 📞 **도움말: FAQ**

**Q: 카카오페이로 쿠팡 2번 주문했는데, 카드사에는 1줄로 나왔어요**
```
A: 금액 대조 안 맞을 수 있습니다.
   - 가격 변동 또는 프로모션 적용 가능
   - 수동으로 메모 추가하기
   - 또는 각 주문을 별도로 분류
```

**Q: 카카오페이 거래내역서 다운로드를 못 했어요**
```
A: 카카오페이 고객센터(챗봇) 이용
   1. "증명서 발급" 검색
   2. "거래내역서 발송" 선택
   3. 이메일로 수령
   4. CSV 파일 직접 업로드 가능
```

**Q: 분해 후 실수로 하나를 또 추가했어요**
```
A: 원본 거래가 숨겨져 있으므로 중복이 되지 않습니다.
   하지만 원상복구하려면 분해 취소 버튼을 누르세요.
```

**Q: 사업비와 생활비가 섞여 있는데, 분류하는 법은?**
```
A: 쿠팡 분해기 사용 시 각 상품마다 "비즈니스" 토글 설정
   - 노트북: [비즈니스] ON  → 개인 통계 제외
   - 우유: [비즈니스] OFF → 개인 통계 포함
```
